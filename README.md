# Storefront

Storefront is a modern, premium Product Management and E-commerce Dashboard built with Spring Boot, Thymeleaf, and JPA.

## Features
- **Modern Dashboard**: Real-time statistics and analytics.
- **Product Management**: Complete CRUD operations for your catalog.
- **Order Tracking**: Manage user orders and revenue.
- **Security**: Robust authentication and authorization.
- **Premium UI**: Sleek dark-mode design with glassmorphism and micro-animations.

## Technology Stack
- **Backend**: Spring Boot 3.2, Spring Security, Spring Data JPA
- **Frontend**: Thymeleaf, Vanilla CSS, Inter Typography
- **Database**: MySQL
- **Documentation**: Swagger OpenAPI
